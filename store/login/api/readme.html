Login API

Format

The API uses POST (not any other HTTP method). Send a request like so:
http -f POST localhost/login/api/api.php action=[ACTION] auth_token=[AUTH_TOKEN] other_options=[OPTIONS]
Check below for actions that can be executed with the API.

Actions

All of the below actions are designed to be included in the ACTION field when POSTing.
  • create_usr
    Creates a new user.
    Authentication : None.
    Required fields: auth_token, name.
  • delete_usr
    Delete a specific user.
    Authentication : Must be admin.
    Required fields: auth_token, username, password.
  • delete_all_usrs
    Deletes all users.
    Authentication : Must be admin.
    Required fields: username, password.
